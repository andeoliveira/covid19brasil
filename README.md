## Iniciar e rodar o projeto

Faça o clone do projeto, extraia em um diretório, acesse o mesmo e execute os comandos

```
npm install 
ng serve
```

## Tecnologias utilizadas

Angular 9 (Angular CLI), Typescript, SCSS, HTML5.
