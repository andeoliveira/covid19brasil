Covid-19 Brasil

Os dados são atualizados diáriamente pelos serviços governamentais.

Para o desenvolvimento deste projeto foram utilizados os dados oficiais do governo do Brasil e do projeto Covid19 do Johns Hopkins CSSE. Esses dados estão disponíveis pelo acesso via API's, os quais você pode consultar nos links abaixo:

Brasil IO Covid19API

Esse projeto segue a licença open-source MIT e tem referência do projeto Covid19India.org o qual está disponível no github sob a licença MIT, para mais detalhes acesso o link abaixo: Covid19 India
